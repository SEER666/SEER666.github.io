今天的学习内容

今天我学习了如何使用 Hexo 来创建博客，并将其部署到 GitHub Pages 上。以下是一些我学习到的重要内容：

1. Hexo 的基本操作

  • 创建新文章：使用 hexo new <title> 来创建新文章。
  • 生成静态文件：使用 hexo generate 来生成静态文件。
  • 本地预览：使用 hexo server 启动本地服务器进行预览。
